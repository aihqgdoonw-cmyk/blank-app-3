# 🧩 받아올림·받아내림 블록 학습 앱

> 초등학교 3~4학년 학생들이 받아올림과 받아내림의 원리를 **시각적으로 이해**하는 대화형 학습 앱입니다.

![Python](https://img.shields.io/badge/python-3.8+-green)
![Streamlit](https://img.shields.io/badge/streamlit-1.0+-red)

---

## 🎯 핵심 개념

| 개념 | 설명 |
|------|------|
| **받아올림** | 일의 자리에서 10개가 모이면 → 한 자리 위로 올려줌 |
| **받아내림** | 일의 자리가 부족하면 → 열 자리에서 빌려옴 |
| **블록 교환** | 1블록 10개 ↔ 10블록 1개 직접 조작 |

---

## ✨ 주요 기능

### 🎮 블록 시스템
- **1 블록** (🟨) — 일의 자리 표현
- **10 블록** (🟩) — 십의 자리 표현  
- **100 블록** (🟦) — 백의 자리 표현

### 🔄 블록 교환소
학생이 직접 버튼을 눌러 교환합니다:
- 1블록 10개 → 10블록 1개
- 10블록 1개 → 1블록 10개
- 10블록 10개 → 100블록 1개

### 📚 학습 모드
- **받아올림 전용** — 덧셈에서 자리 올림 이해
- **받아내림 전용** — 뺄셈에서 자리 내림 이해
- **혼합 문제** — 둘 다 섞여 있는 문제

### 🎮 게임 요소
- ⭐ **경험치** — 정답 시 20~35점 획득
- 📊 **레벨 시스템** — 경험치 100점마다 레벨 상승
- 🎖️ **배지 시스템** — 특정 조건 달성 시 배지 획득
  - 첫 번째 도전
  - 연속 3회 정답
  - 블록 교환왕 (교환 5회)
  - 레벨 업
  - 원리 탐구자 (10회 정답)
- 🔥 **연속 정답 보너스** — 정답 스트릭 유지 시 추가 경험치

### 🎨 초등학생 친화형 디자인
- 밝고 친근한 색상 사용
- 큼직한 버튼 (터치 최적화)
- 모바일·태블릿 지원

## 📖 사용 방법

### 받아올림 배우기 (예: 27 + 15)
1. 두 수의 블록을 작업대에 놓음 (블록 시스템이 자동 배치)
2. 일의 자리 블록 합계 확인 (2 + 5 = 7개)
3. 다음 문제로 일의 자리가 10개 이상이 되도록 하기 (예: 8 + 7 = 15)
4. **받아올림 발생!** 1블록 10개를 10블록으로 교환
5. **원리 이해**: "10개가 모이니까 한 자리 위로 올라가네!"

### 받아내림 배우기 (예: 52 - 18)
1. 큰 수의 블록을 작업대에 놓음
2. 작은 수를 빼기 시작
3. 일의 자리 블록이 부족하면 **받아내림 필요!**
4. 10블록 1개를 1블록 10개로 교환 (`10블록 1개 → 1블록 10개` 버튼)
5. 이제 충분한 블록으로 뺄셈 완료
6. **원리 이해**: "부족할 때는 한 자리 위에서 빌려오면 된다!"

## 🎓 학습 흐름

```
새 문제 생성
    ↓
블록 자동 배치 (또는 수동 조작)
    ↓
문제 풀이
    ├→ 받아올림/받아내림 필요?
    │   └→ 블록 교환소에서 직접 교환
    ├→ 블록 더하기/빼기
    └→ 최종 답 확인
    ↓
정답 판정 + 경험치 획득
    ↓
배지/레벨 업 확인
    ↓
다음 문제로...
```
